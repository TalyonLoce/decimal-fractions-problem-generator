# Интерактивный курс по математике — Десятичные дроби

Одностраничное веб-приложение для отработки навыков действий с десятичными дробями.  
Генерирует случайные примеры, проверяет ответы и даёт обратную связь.

## Возможности

- 4 раздела: **сложение**, **вычитание**, **умножение**, **деление** десятичных дробей.
- Теоретическая справка с правилами и примерами в каждом разделе.
- Генерация неограниченного количества примеров с разумными диапазонами чисел.
- Переключение между темами без перезагрузки страницы.

## Технологии

- **HTML5** — семантическая разметка.
- **CSS3** — адаптивная вёрстка, Flexbox, медиазапросы, анимации.
- **JavaScript (ES6+)** — генерация чисел, управление состоянием, проверка ответов, обработка событий.

## Как запустить

1. Скачайте все файлы проекта в одну папку.
2. Откройте файл `index.html` в любом современном браузере.
3. Наслаждайтесь тренировкой! Никаких серверов и дополнительных зависимостей не требуется.

Или посмотрите онлайн:  
`https://talyonloce.github.io/decimal-fractions-problem-generator/`

## Как это работает

- При загрузке страницы генерируются первые примеры для всех четырёх разделов.
- Вы переключаете раздел — автоматически генерируется новый пример.
- Вводите ответ, нажимаете «Проверить» (или клавишу Enter) — получаете сообщение о правильности.
- Кнопка «Новый пример» создаёт другое случайное выражение в текущем разделе.
